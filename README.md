# CRUD Fashion Store

## Deskripsi
Project ini merupakan aplikasi berbasis web menggunakan Laravel yang menerapkan konsep CRUD (Create, Read, Update, Delete) dengan tema Fashion Store.

## Fitur
- Manajemen Kategori
- Manajemen Brand
- Manajemen Produk

## Teknologi
- Laravel 12
- PHP
- MySQL

## Cara Instalasi
1. Clone repository
2. composer install
3. copy .env.example ke .env
4. setting database
5. php artisan key:generate
6. php artisan migrate
7. php artisan serve
